# Deploiement du modèle de classification de plantes


### 🎯 *Objectif du module*


Afin de pouvoir utiliser le modèle de classification de plantes que j'ai entrainé dans les étapes précédentes, j'ai décidé de le déployer sur une API pour permettre à tous et chacun de l'utiliser pour faire la classification d'images de plantes. Cette étape de déploiement est cruciale pour rendre le modèle accessible et utilisable par un large public, et pour permettre à d'autres personnes de bénéficier des résultats de mon travail.
