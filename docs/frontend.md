🎯 Objectif du module

L'objectif de ce module est de déployer le modèle de classification de plantes que j'ai entrainé dans les étapes précédentes, pour permettre à tous et chacun de l'utiliser pour faire la classification d'images de plantes. Pour cela, j'ai mis en ligne une interface utilisateur en utilisant Streamlit, qui permet une utilisation facile et intuitive du modèle.
